# Personal Expense Tracker (Java)

## Description
This project is a simple command-line based Personal Expense Tracker developed using Java. It helps users manage and track their daily expenses efficiently.

## Problem Statement
Students often face difficulty in tracking their expenses, which leads to poor financial management. This project provides a simple solution to record, view, and analyze expenses.

## Features
- Add Expense (title, amount, category)
- View All Expenses
- Calculate Total Expense
- Search Expenses by Category
- Data stored using file handling (expenses.txt)

## Technologies Used
- Java
- File Handling
- Object-Oriented Programming (OOP)
